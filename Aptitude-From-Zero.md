# Aptitude & Reasoning — Learn From Zero (Complete Beginner Guide)

## 🤔 What is Aptitude?

**You DON'T need advanced math** — just basic arithmetic (+, -, ×, ÷), percentages, and logical thinking.

## 📊 Chapter 1: Percentages (START HERE)

### What is a Percentage?
"Per cent" means "per hundred". 25% means 25 out of 100.

### Converting
| Fraction | Decimal | Percentage |
|----------|---------|------------|
| 1/2 | 0.5 | 50% |
| 1/4 | 0.25 | 25% |
| 1/5 | 0.2 | 20% |
| 1/10 | 0.1 | 10% |
| 1/100 | 0.01 | 1% |

### How to Calculate Percentage of a Number
**Formula**: (Percentage / 100) × Number

**Example**: What is 20% of 150?
```
= (20/100) × 150
= 0.2 × 150
= 30
```

**Example**: What is 15% of 200?
```
= (15/100) × 200
= 0.15 × 200
= 30
```

### Shortcut: Think in Fractions
Instead of calculating 25% of 80, think:
25% = 1/4, so 80 ÷ 4 = **20**

| Shortcut | How to Think |
|----------|-------------|
| 50% | Half (÷2) |
| 25% | Quarter (÷4) |
| 20% | One-fifth (÷5) |
| 10% | One-tenth (÷10) |
| 5% | Half of 10% |
| 1% | One-hundredth (÷100) |

### Percentage Increase/Decrease
**Formula**: ((New - Old) / Old) × 100

**Example**: Price went from ₹200 to ₹250. What is % increase?
```
= ((250 - 200) / 200) × 100
= (50 / 200) × 100
= 25%
```

**Example**: Price went from ₹500 to ₹400. What is % decrease?
```
= ((500 - 400) / 500) × 100
= (100 / 500) × 100
= 20%
```

### Reverse: Finding Original Number
**If x% of a number is Y, what is the number?**
Formula: Number = Y × (100/x)

**Example**: 20% of a number is 40. What is the number?
```
= 40 × (100/20)
= 40 × 5
= 200
```

### Practice Problems
1. What is 30% of 240? → 72
2. 60 is what % of 240? → 25%
3. A number increased from 80 to 100. % increase? → 25%
4. 15% of a number is 45. What is the number? → 300

## 💰 Chapter 2: Profit & Loss

### Key Terms
- **Cost Price (CP)**: Price at which you BUY
- **Selling Price (SP)**: Price at which you SELL
- **Profit**: When SP > CP (you earned money)
- **Loss**: When SP < CP (you lost money)
- **Marked Price (MP)**: Price before discount

### Basic Formulas
```
Profit = SP - CP
Loss = CP - SP
Profit % = (Profit / CP) × 100
Loss % = (Loss / CP) × 100
```

### ALWAYS calculate percentage on COST PRICE!

### Example 1: Simple Profit
You bought a phone for ₹10,000 and sold it for ₹12,000.
```
Profit = 12000 - 10000 = ₹2000
Profit % = (2000 / 10000) × 100 = 20%
```

### Example 2: Simple Loss
You bought a book for ₹500 and sold it for ₹400.
```
Loss = 500 - 400 = ₹100
Loss % = (100 / 500) × 100 = 20%
```

### Finding SP from CP and Profit%
```
SP = CP × (100 + Profit%) / 100
```
**Example**: CP = ₹800, Profit = 25%. Find SP.
```
SP = 800 × (100 + 25) / 100
SP = 800 × 125 / 100
SP = ₹1000
```

### Finding CP from SP and Profit%
```
CP = SP × 100 / (100 + Profit%)
```
**Example**: SP = ₹600, Profit = 20%. Find CP.
```
CP = 600 × 100 / (100 + 20)
CP = 600 × 100 / 120
CP = ₹500
```

### Discount
**Discount** = Marked Price - Selling Price
**Discount %** = (Discount / Marked Price) × 100

**Example**: Marked price = ₹1000, Sold for ₹800.
```
Discount = 1000 - 800 = ₹200
Discount % = (200 / 1000) × 100 = 20%
```

### Practice Problems
1. CP = ₹500, SP = ₹600. Profit%? → 20%
2. CP = ₹800, Loss = 10%. SP? → ₹720
3. SP = ₹900, Profit = 20%. CP? → ₹750
4. MP = ₹1200, Discount = 25%. SP? → ₹900

## ⏱️ Chapter 3: Time & Work

### Concept
If A can do a work in N days, then A does 1/N of the work per day.

### Example: Single Person
Ram can complete a work in 10 days. How much work does he do in 1 day?
```
Work per day = 1/10
```

### Example: Working Together (THE LCM METHOD!)
A can do work in 12 days, B in 18 days. Together?

**Step 1**: Find LCM of 12 and 18
```
12 = 2×2×3
18 = 2×3×3
LCM = 2×2×3×3 = 36
```

**Step 2**: LCM = Total Work (think of it as 36 units)
```
A's work per day = 36/12 = 3 units
B's work per day = 36/18 = 2 units
Together per day = 3 + 2 = 5 units
```

**Step 3**: Time = Total Work / Work per day
```
Time = 36 / 5 = 7.2 days
```

### Another Example
A = 6 days, B = 8 days, C = 12 days. Together?
```
LCM(6, 8, 12) = 24
A = 24/6 = 4/day
B = 24/8 = 3/day
C = 24/12 = 2/day
Together = 4+3+2 = 9/day
Time = 24/9 = 8/3 = 2.67 days
```

### If A Leaves After Some Days
A and B work together for 2 days, then A leaves. How long does B take to finish?

**Example**: A = 10 days, B = 15 days. Work together 2 days, then B finishes.
```
LCM(10, 15) = 30
A = 3/day, B = 2/day
Together = 5/day
In 2 days = 10 units done
Remaining = 30 - 10 = 20 units
B finishes in = 20/2 = 10 more days
Total = 2 + 10 = 12 days
```

### Practice Problems
1. A = 8 days, B = 12 days. Together? → 4.8 days
2. A = 5 days, B = 10 days, C = 15 days. Together? → 30/11 = 2.73 days
3. A = 6 days, B = 8 days. Together 2 days, then A leaves. B finishes? → 10/3 more days

## ⏰ Chapter 4: Time, Speed & Distance

### Basic Formula
```
Speed = Distance / Time
Distance = Speed × Time
Time = Distance / Speed
```

### Units!
- Speed in km/hr → Distance in km, Time in hours
- Speed in m/s → Distance in meters, Time in seconds

### Converting km/hr to m/s
```
1 km/hr = 1000m / 3600s = 5/18 m/s
km/hr × 5/18 = m/s
m/s × 18/5 = km/hr
```

**Example**: 72 km/hr in m/s?
```
72 × 5/18 = 20 m/s
```

### Example 1: Basic
A car travels 120 km in 2 hours. What is speed?
```
Speed = 120 / 2 = 60 km/hr
```

### Example 2: Finding Distance
A man walks at 5 km/hr for 3 hours. Distance?
```
Distance = 5 × 3 = 15 km
```

### Example 3: Relative Speed (Same Direction)
Two cars moving same direction at 60 and 40 km/hr.
```
Relative Speed = 60 - 40 = 20 km/hr
```

### Example 4: Relative Speed (Opposite Direction)
Two cars moving towards each other at 60 and 40 km/hr.
```
Relative Speed = 60 + 40 = 100 km/hr
```

### Average Speed (Equal Distance)
If you travel equal distances at speeds x and y:
```
Average Speed = 2xy / (x + y)
```

**Example**: Travel 60 km at 30 km/hr and return at 60 km/hr.
```
Avg Speed = 2 × 30 × 60 / (30 + 60)
= 3600 / 90
= 40 km/hr
```
NOT (30+60)/2 = 45! That's wrong!

### Train Problems
**Time to cross a pole** = Length of train / Speed
**Time to cross platform** = (Train + Platform) / Speed

**Example**: Train 150m long, speed 20 m/s, crosses a pole.
```
Time = 150 / 20 = 7.5 seconds
```

### Practice Problems
1. 180 km in 3 hours. Speed? → 60 km/hr
2. 90 km/hr = ? m/s → 25 m/s
3. Train 200m long, 300m platform, speed 25 m/s. Time? → 20 seconds

## 🔢 Chapter 5: Averages

### What is Average?
Average = Sum of all values / Number of values

### Example
Find average of 10, 20, 30, 40, 50.
```
Sum = 10 + 20 + 30 + 40 + 50 = 150
Count = 5
Average = 150 / 5 = 30
```

### Finding Missing Value
Average of 5 numbers is 20. Four numbers are 15, 18, 22, 25. Find the fifth.
```
Total Sum = Average × Count = 20 × 5 = 100
Sum of 4 numbers = 15 + 18 + 22 + 25 = 80
Missing = 100 - 80 = 20
```

### When New Member Joins
Old average of 4 numbers = 15. New member's value = 25. New average?
```
Old sum = 15 × 4 = 60
New sum = 60 + 25 = 85
New average = 85 / 5 = 17
```

### Shortcut: Effect of Adding/Removing
- Adding a number HIGHER than average → average increases
- Adding a number LOWER than average → average decreases

### Practice Problems
1. Average of 8, 12, 16, 20? → 14
2. Average of 5 numbers is 30. Sum? → 150
3. Average age of 3 people is 20. 4th person is 28. New average? → 22

## 🧠 Chapter 6: Reasoning — Number Series

### What is a Series?
A sequence of numbers following a pattern. Find the pattern → predict the next number.

### Step-by-Step Method
1. Find the DIFFERENCE between consecutive numbers
2. If differences aren't constant, find differences of differences
3. Look for multiplication/division patterns
4. Check if numbers are squares/cubes

### Example 1: Addition Pattern
**2, 5, 8, 11, 14, ?**
```
Differences: 3, 3, 3, 3, ?
Next difference = 3
Answer = 14 + 3 = 17
```

### Example 2: Increasing Difference
**2, 6, 12, 20, 30, ?**
```
Differences: 4, 6, 8, 10, ?
Pattern: +2 each time
Next difference = 12
Answer = 30 + 12 = 42
```

### Example 3: Multiplication
**2, 6, 18, 54, ?**
```
Ratios: 6/2=3, 18/6=3, 54/18=3
Pattern: ×3 each time
Answer = 54 × 3 = 162
```

### Example 4: Square Pattern
**1, 4, 9, 16, 25, ?**
```
1=1², 4=2², 9=3², 16=4², 25=5²
Next = 6² = 36
```

### Example 5: Fibonacci Pattern
**1, 1, 2, 3, 5, 8, ?**
```
Each number = sum of previous two
1+1=2, 1+2=3, 2+3=5, 3+5=8
Next = 5+8 = 13
```

### Example 6: Alternating Pattern
**2, 7, 4, 9, 6, 11, ?**
```
Odd positions: 2, 4, 6, ? → +2 each
Even positions: 7, 9, 11 → +2 each
Next (odd position) = 6 + 2 = 8
```

### Practice Problems
1. 3, 6, 9, 12, ? → 15
2. 1, 3, 6, 10, 15, ? → 21
3. 2, 4, 8, 16, ? → 32
4. 1, 8, 27, 64, ? → 125 (cubes)
5. 5, 10, 20, 40, ? → 80

## 🧠 Chapter 7: Reasoning — Blood Relations

### Common Relations
```
Father's father = Grandfather
Father's mother = Grandmother
Mother's father = Maternal Grandfather
Mother's mother = Maternal Grandmother
Father's brother = Uncle
Father's sister = Aunt
Mother's brother = Uncle
Mother's sister = Aunt
Brother's son = Nephew
Brother's daughter = Niece
Sister's son = Nephew
Sister's daughter = Niece
Uncle's son = Cousin
Uncle's daughter = Cousin
```

### Strategy: DRAW A FAMILY TREE!
Use symbols:
- □ = Male
- ○ = Female
- ─ = Married
- │ = Child

### Example
"Pointing to a man, a woman said: His mother is the only daughter of my mother."

**Step by Step:**
1. "Only daughter of my mother" = the woman herself
2. So: "His mother is [the woman]"
3. Answer: The woman is the man's **MOTHER**

### Another Example
"A is B's brother. C is A's mother. D is C's father."

Draw it:
```
D (grandfather) ─ C (mother)
                  │
            A (brother) ─ B
```
B is D's **grandson/granddaughter**

### Practice Problems
1. "He is the son of my mother's son." → He is my nephew
2. Pointing to a photo: "She is the daughter of my grandfather's only son." → She is my sister

## 🧠 Chapter 8: Reasoning — Direction Sense

### Remember This!
```
        N
      NW  NE
    W   •   E
      SW  SE
        S
```

### Clockwise: N → E → S → W
### Counter-clockwise: N → W → S → E

### Right Turn = 90° Clockwise
### Left Turn = 90° Counter-clockwise

### Example
A man walks 5 km North, then turns right and walks 3 km. Where is he?

```
Start → 5km North → Turn Right (East) → 3km East

Final position: 5km North, 3km East of start
Distance from start = √(5² + 3²) = √(25+9) = √34 km
Direction from start: North-East
```

### Shadow Trick
- **Morning (sunrise)**: Shadow falls WEST
- **Evening (sunset)**: Shadow falls EAST
- **Noon**: Shadow falls towards shortest direction

### Example
Evening time, a man's shadow falls to his right. Which direction is he facing?
```
Evening → Shadow is East
Shadow is to his right → He's facing North
```

### Practice Problems
1. Walk 3m North, 4m East. Distance from start? → 5m
2. Facing South, turn right. Now facing? → West
3. Morning, shadow falls? → West
