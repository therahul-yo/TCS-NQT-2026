## 💻 Coding Questions — What to Expect

### Common Question Types

**Arrays:**
- Find max/min element
- Reverse array
- Rotate array by K
- Find second largest
- Count even/odd numbers
- Sum of array elements

**Strings:**
- Check palindrome
- Count vowels and consonants
- Reverse string
- Find most frequent character
- Remove spaces/special characters
- Check anagram

**Numbers:**
- Check prime
- Print fibonacci series
- Factorial
- GCD/LCM
- Armstrong number
- Sum of digits

**Sorting/Searching:**
- Binary search
- Sort array (any method)
- Find element in sorted array

### Difficulty Level
- **Easy**: 40% of questions
- **Medium**: 45% of questions
- **Hard**: 15% of questions

### Language Tips
- **Python** is easiest for beginners (shorter code)
- **Java** is most commonly used by TCS developers
- **C/C++** is fastest but more verbose

## ⏱️ Time Management Strategy

### Foundation Section (75 mins, 65 questions)
- **Target**: 1 min 10 sec per question
- **Order**: Easy topics first (series, vocab, basic math)
- **Skip**: Spend more than 1.5 min? Flag and move on
- **Last 5 mins**: Go back to flagged questions

### Advanced Section (115 mins)
- **Advanced MCQs**: 25 mins for 14-16 questions
- **Coding**: 90 mins for 2 questions
  - **Q1 (Easy)**: 30-40 mins
  - **Q2 (Medium)**: 40-50 mins
  - **Buffer**: 10 mins for testing/debugging

### Coding Time Breakdown
1. Read problem & understand: 5 mins
2. Plan approach: 5 mins
3. Write code: 15-20 mins
4. Test with examples: 5-10 mins
5. Debug if needed: 5-10 mins
