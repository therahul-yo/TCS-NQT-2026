# Advanced Quantitative Ability — Advanced Shortcuts & Tricks

## 📊 Data Interpretation (DI)

### Types
1. **Tables** — Read carefully, practice addition/subtraction
2. **Bar Charts** — Compare lengths visually
3. **Pie Charts** — Total = 360°, each % = 3.6°
4. **Line Graphs** — Track trends, calculate slopes
5. **Mixed Charts** — Combine 2+ types

### Pie Chart Shortcut
- **Value = (Percentage/100) × Total**
- **Angle = (Value/Total) × 360°**
- To find value from angle: Value = (Angle/360) × Total

### Speed Tips for DI
1. **Approximate** — round numbers to nearest 10/100
2. **Use ratios** instead of exact calculations
3. **Estimate percentages** using 10%, 5% benchmarks
4. **Read questions first** — know what to calculate
5. **Common fractions**: 1/4=25%, 1/3=33.33%, 1/5=20%, 1/8=12.5%

## 🎲 Probability

### Key Formulas
- **P(A) = Favorable outcomes / Total outcomes**
- **P(not A) = 1 - P(A)**
- **P(A and B) = P(A) × P(B)** [independent events]
- **P(A or B) = P(A) + P(B) - P(A and B)**

### Coin & Dice
- **1 coin**: P(H)=1/2, P(T)=1/2
- **2 coins**: P(both H)=1/4, P(at least one H)=3/4
- **1 die**: P(any number)=1/6
- **2 dice**: Total outcomes = 36
- P(sum 7) = 6/36 = 1/6 (max probability for any sum)

### Cards
- **52 cards**: 4 suits × 13 cards
- P(any specific card) = 1/52
- P(any Ace) = 4/52 = 1/13
- P(any Heart) = 13/52 = 1/4

## 🎯 Permutations & Combinations

### Key Formulas
- **P(n,r) = n! / (n-r)!** (order matters)
- **C(n,r) = n! / [r! × (n-r)!]** (order doesn't matter)
- **C(n,r) = C(n, n-r)**
- **C(n,0) + C(n,1) + ... + C(n,n) = 2^n**

### Shortcut: nCr Values
```
n=1: 1
n=2: 1 2 1
n=3: 1 3 3 1
n=4: 1 4 6 4 1
n=5: 1 5 10 10 5 1
```

### Common Problems
- **Arranging n items**: n!
- **Circular arrangement**: (n-1)!
- **Arrangement with identical items**: n! / (p! × q!)
- **Choosing r from n**: C(n,r)

## ⏰ Clocks & Calendars

### Clock Shortcuts
- **Minute hand**: 360° in 60 min = 6°/min
- **Hour hand**: 360° in 12 hrs = 0.5°/min
- **Relative speed**: 5.5°/min (minute hand gains on hour hand)
- **Hands coincide**: Every 65 + 5/11 minutes
- **Hands at right angle**: Every 32 + 8/11 minutes
- **Hands opposite**: Every 65 + 5/11 minutes

### Calendar
- **Leap year**: Divisible by 4 (but not 100 unless also 400)
- **Odd days**: Days remaining after complete weeks
  - Ordinary year = 1 odd day
  - Leap year = 2 odd days
- **100 years**: 5 odd days
- **200 years**: 3 odd days
- **300 years**: 1 odd day
- **400 years**: 0 odd days

### Day Calculation
- 0 odd days = Sunday
- 1 odd day = Monday
- 2 odd days = Tuesday
- ...and so on

## 📐 Geometry

### Key Formulas
- **Triangle Area**: ½ × base × height
- **Heron's Formula**: √[s(s-a)(s-b)(s-c)] where s=(a+b+c)/2
- **Circle**: Area = πr², Circumference = 2πr
- **Rectangle**: Area = l×b, Perimeter = 2(l+b)
- **Square**: Area = a², Perimeter = 4a, Diagonal = a√2

### Pythagorean Triplets (MEMORIZE!)
- 3,4,5 | 5,12,13 | 7,24,25 | 8,15,17 | 9,40,41
- 6,8,10 | 10,24,26 | 12,16,20

### Quick Tricks
- **Diagonal of rectangle** = √(l² + b²)
- **Area of equilateral triangle** = (√3/4) × a²
- **Volume of cube** = a³, Surface area = 6a²
- **Volume of cylinder** = πr²h

## 🔢 Elementary Statistics

### Key Formulas
- **Mean** = Sum / Count
- **Median** = Middle value (or average of two middle values)
- **Mode** = Most frequent value
- **Range** = Max - Min
- **Variance** = Σ(xᵢ - mean)² / n
- **Standard Deviation** = √Variance

### Shortcut: Mean of Combined Groups
If group A has mean M₁ with n₁ items, group B has mean M₂ with n₂ items:
- **Combined Mean** = (n₁M₁ + n₂M₂) / (n₁ + n₂)

### Shortcut: Effect of Adding Constant
- If every value +k, New mean = Old mean + k
- If every value ×k, New mean = Old mean × k
- SD remains same when adding constant
- SD becomes ×k when multiplying by k

## 📏 Ages Problems

### Strategy
1. **Define variables** for present ages
2. **Use "years ago"** and "years hence" to set up equations
3. **Key relationship**: (Age after n years) = (Present age) + n

### Shortcut: Ratio Method
If ages are in ratio a:b and sum is S:
- Person 1 = S × a/(a+b)
- Person 2 = S × b/(a+b)

### If age ratio changes over time
- (a+n)/(b+n) = p/q → Solve for n

## ⚖️ Equations

### Simultaneous Equations Shortcut
For system: ax + by = c, dx + ey = f
- x = (ce - bf) / (ae - bd)
- y = (af - cd) / (ae - bd)

### Quadratic Formula
For ax² + bx + c = 0:
- x = (-b ± √(b² - 4ac)) / 2a
- **Discriminant** (b² - 4ac): >0 → 2 real roots, =0 → 1 root, <0 → no real roots
