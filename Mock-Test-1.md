### Section 2: Coding (2 questions, 90 minutes)

#### Coding Question 1 (Easy-Medium)
**Problem**: Given an array of integers, find the second largest element without sorting.

**Input**: First line contains N (size of array). Second line contains N space-separated integers.

**Output**: Print the second largest element.

**Example**:
```
Input:
5
12 35 1 10 34

Output:
34
```

**Constraints**: 2 ≤ N ≤ 10⁵, -10⁹ ≤ arr[i] ≤ 10⁹

#### Coding Question 2 (Medium)
**Problem**: Given a string, find the first non-repeating character. If all characters repeat, print -1.

**Input**: A single string S.

**Output**: The first non-repeating character, or -1.

**Example**:
```
Input:
programming

Output:
p
```

**Constraints**: 1 ≤ |S| ≤ 10⁵

### Coding Solutions
```python
# Q1: Second Largest
n = int(input())
arr = list(map(int, input().split()))
first = second = float('-inf')
for num in arr:
    if num > first:
        second = first
        first = num
    elif num > second and num != first:
        second = num
print(second)

# Q2: First Non-Repeating Character
from collections import Counter
s = input()
freq = Counter(s)
for char in s:
    if freq[char] == 1:
        print(char)
        break
else:
    print(-1)
```
